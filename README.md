## Run configurations

### 📦 Run with docker(make):
``make d-homework-i-run``

### 🚮 Purge after run(make):
``make d-homework-i-purge``

### 🚫 Run without docker(make):
``make homework-i-run``

### 📋 Run with script:
``sh run.sh``

### 🖐 Run manually:
```
python3 -m venv venv &&
pip install --upgrade pip &&
pip install --requirement requirements.txt &&
python main.py
```
